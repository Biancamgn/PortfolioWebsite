JavaScript Code

let box = document.getElementById("box");
let position = box.getBoundingClientRect();

console.log("left:", position.left);
console.log("top:", position.top);
console.log("right:", position.right);
console.log("bottom:", position.bottom);
console.log("width:", position.width);
console.log("height:", position.height);

getBoundingClientRect() returns a DOMRect object with the size of an element and its position relative to the viewport.
