Bouncing Ball Animation

A ball that bounces off walls using setInterval() for continuous animation

Interactive Demo

X Position 0
Y Position 0
Bounces 0
Speed 4

JavaScript Code

let x = 0, y = 0;
let dx = 5, dy = 3;
const ball = document.getElementById("ball");
const arena = document.getElementById("ballArena");

setInterval(() => {
  x += dx;
  y += dy;

  // Bounce off walls
  if (x + 30 >= arena.clientWidth || x <= 0) dx *= -1;
  if (y + 30 >= arena.clientHeight || y <= 0) dy *= -1;

  ball.style.left = x + "px";
  ball.style.top = y + "px";
}, 16);

setInterval() calls the function repeatedly at a fixed interval (16ms ≈ 60fps), creating smooth animation by updating the ball's position each frame.
